\section{Experiment=25: convection in a box, BA vs EBA}

This experiment is very similar to the convection experiment of \textcite{blbc89}.
Free slip boundary conditions are applied on all sides.
The main difference resides in the fact that all physical parameters are Earth-like:

\begin{lstlisting}
Lx = 3000e3 # m
Lz = 3000e3 # m
Ttop = 0 +273 # K
Tbottom = 3000 + 273 # K
alphaT = 1e-5  # /T
hcond = 3  
hcapa = 1200 #
rho0 = 2500 # kg/m3
g0 = 10 # m/s2
eta0 = 2.025e24 # Pa.s
\end{lstlisting}
By choosing these parameters we can make sure that the Rayleigh number, the 
thermal diffusivity and the dissipation number are assigned the desired value:
\begin{lstlisting}
Di = alphaT * g0 * Lz / hcapa
kappa = hcond / rho0 / hcapa
Ra = alphaT * rho0 * g0 * (Lz**3) * (Tbottom - Ttop) / kappa / eta0
\end{lstlisting}
which yields
\begin{verbatim}
-> Di= 0.25
-> kappa= 1e-06
-> Ra= 10000.000000000002
\end{verbatim}

In all the following figures time is rendered dimensionless by diving it by the ref time. 
Reference velocity and pressure values are also taken from \textcite{kilv10}:
\begin{lstlisting}
reftime=rho0*hcapa*Lz**2/hcond 
refvel=Lz/reftime 
refPress = eta0 * hcond / rho0 / hcapa / Lz**2
\end{lstlisting}
which gives (for $Ra=10^4$):
\begin{verbatim}
-> reftime 9.000000e+18 s | 2.851928e+11 yr
-> refvel 3.333333e-13 m/s | 1.051920e-03 cm/yr
-> refPress 2.250000e+05 
\end{verbatim}

I have also built this experiment with ASPECT and the results obtained with 
MEEUUW are compared with those of ASPECT in what follows.

\subsection{EBA formulation}

ok so i have a pb: 
the adiabatic term contains T, so that the value of the temperature
matters a lot. I then observe differences wrt measurements whether i set Ttop
at 0, 273 or 293K.  
Basically by changing Ttop, but making sure there is a 3000K difference I can 'choose' or 'tune'
the value of Nu and vrms.
Assuming there is no mistake in the implementation, how can I choose Ttop?!

After a while I found lee\_\_13 paper which does exactly what I was aiming to do:
a non-dimensionless King et al setup. AND they also show/indicate T=273K top
and 3273K bottom. HOWEVER, I still do not get close to the expected values.
